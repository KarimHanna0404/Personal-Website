Adaptive Priority Queue (APQ) in Java

Developed an Adaptive Priority Queue (APQ) in Java using a min-heap combined with a position map to support dynamic key updates and efficient access. Implemented core operations such as insertion (O(log n)), removal of the minimum element (O(log n)), and key updates (O(log n)), while maintaining fast element lookups in O(1) using a hash map. Designed and tested the structure through a custom driver program simulating priority changes in real-time task management scenarios.

Technologies: Java, Min-Heap, HashMap, Object-Oriented Programming (OOP), Time Complexity Analysis, Data Structures, Collections Framework (for optional utilities)
